RE:WIRE

This form must be used in case you've already been assigned a Friend Code in-game before using this service.

To check if you have been assigned a Friend Code go to the Top Menu, then "Others", "Nintendo Wi-Fi Connection", "My Friend Code"

This lets you rewire your Friend Code to your new profile on this server if it hasn't been already used.

Follow the steps below:

  1. Go to your Nintendo DS settings and change your user name to something unique (best is to fill with random characters)
  2. Go to your game and rename your team to something unique (best is to fill with random characters)
  3. Put the newly chosen user names and team names
  4. Put your assigned Friend Code
  5. Submit
  6. Connect to the Wi-Fi server (the easiest way is to check SOS Mails on the server)
  7. You can now change back your user name and team name to their original settings
